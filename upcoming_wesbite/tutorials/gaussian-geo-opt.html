Running a Geometry Optimization with Gaussian

This tutorial walks through the process of setting up and running a basic ground-state geometry optimization for an Azobenzene molecule using Gaussian. Geometry optimization is a fundamental task in computational chemistry used to find the lowest energy structure of a molecule.

Input File Breakdown

Below is the complete input file (`.gjf` or `.com`). The key components are:

  • Route Section (`#` line): Specifies the type of calculation. opt requests an optimization, wb97xd/6-31g(d) defines the theory level and basis set, and geom=connectivity tells Gaussian to read bond connectivity information from the end of the file.
  • Title Section: A brief, human-readable description of the job.
  • Charge and Multiplicity: `0 1` for a neutral singlet state.
  • Molecular Specification: The XYZ coordinates of each atom.
  • Connectivity: (Optional but recommended) Explicitly defines the bonding, which can help the optimization algorithm.
